# 🎬 Film Hub

## 📌 Why Films?
A good movie finder app is essential for enjoying quality time watching shows or movies, so I decided to code one of my own.
This app makes it easier finding what you like most or discovering some new genres what you might like.

## 🚀 Features
- 🔍 Browse popular movies in a responsive format
- 🧾 Movie details including rating, popularity, and posters
- 📝 Watchlist: Add movies to your personal watchlist to save them for later  


## 🛠️ Tech Stack
- ⚛️ **React** – UI rendering and component-based architecture  
- 🧭 **React Router** – Page navigation and routing  
- 🎨 **Tailwind CSS** – Utility-first styling framework  
- 🌐 **TMDb API** – Source of movie data (REST API)
